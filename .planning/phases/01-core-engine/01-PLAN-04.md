# Plan 04: PDF Generation with Playwright

**Phase:** 1 - Core Engine
**Goal:** Convert HTML to PDF using Playwright

## Steps

1. **Create `src/lib/pdf.ts`**:
   - Initialize Playwright browser (chromium)
   - Create new page and set content
   - Use `page.pdf()` with proper options
   - Support format selection (A4 vs 16:9)
   - Clean up browser instance

2. **PDF options configuration**:
   - A4: `format: 'A4', printBackground: true, margin: { top: '20mm', right: '20mm', bottom: '20mm', left: '20mm' }`
   - 16:9: `width: '1920px', height: '1080px'` for slides
   - Always `printBackground: true` for colors/gradients
   - `preferCSSPageSize: true` to respect @page rules

3. **Create `src/cli.ts`**:
   - Commander.js setup
   - `build` command: `beautifuldocs build <file>`
   - Parse input file, generate PDF to output/
   - Error handling and user feedback

4. **Test end-to-end**: markdown → HTML → PDF

## Success Criteria
- CLI command works: `bun run src/cli.ts build test.md`
- Generated PDF opens in Preview without errors
- Background colors and images render correctly
- Multi-page documents work correctly
