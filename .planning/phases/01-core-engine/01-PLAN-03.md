# Plan 03: HTML Template Engine

**Phase:** 1 - Core Engine
**Goal:** Generate HTML documents with configurable templates

## Steps

1. **Create `src/lib/template.ts`**:
   - Template function that takes parsed content + options
   - Returns complete HTML document with CSS
   - Support for different page formats (A4, 16:9)

2. **Create `src/templates/base.html`**:
   - Base template with CSS variables for theming
   - Page break controls (@media print)
   - Typography reset and base styles
   - Margins optimized for PDF output

3. **Create `src/lib/css.ts`** with CSS utilities:
   - A4 page dimensions (210mm × 297mm)
   - 16:9 slide dimensions (1920px × 1080px)
   - Print-specific CSS (@page rules)

4. **Test template generation** with sample data

## Success Criteria
- Generated HTML is valid and complete
- CSS includes proper @media print rules
- Page dimensions are correct for A4 and 16:9
