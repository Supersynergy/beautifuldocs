# Plan 03: Effects & Gradients Library

**Phase:** 2 - Visual System
**Goal:** CSS-only effects: glassmorphism, gradients, patterns

## Steps

1. **Create src/lib/effects.ts**:
   - Gradient definitions (linear, radial, conic)
   - 50+ prebuilt gradient combos
   - Glassmorphism utility classes
   - Aurora background effect (CSS animation)

2. **Create src/lib/patterns.ts**:
   - Subtle background patterns (dots, grid, topography)
   - CSS-only patterns (gradients + sizing)
   - Noise texture overlay

3. **Add utility classes**:
   - `.gradient-text` - text with gradient fill
   - `.glass` - glassmorphism card
   - `.glow-*` - box-shadow glows
   - `.pattern-*` - background patterns

4. **Template integration**:
   - Each template uses effects appropriately
   - No effects in Minimal template
   - Heavy effects in Neon template

## Success Criteria
- `.gradient-purple-cyan` applies gradient
- `.glass` creates frosted glass effect
- Patterns render at 5% opacity
- Effects work in PDF output
