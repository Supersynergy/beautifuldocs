# Plan 04: Component Library

**Phase:** 2 - Visual System
**Goal:** Rich components: cards, tags, metrics, funnels, progress bars

## Steps

1. **Create src/lib/components.ts**:
   - Card component (with variants: default, outlined, elevated)
   - Tag/Pill component (with color variants)
   - Metric display (big number + label)
   - Progress bar (with percentage)
   - Funnel diagram (simple CSS)
   - Callout/Alert boxes

2. **Markdown syntax**:
   - Use HTML in markdown: `<div class="card">...</div>`
   - Or special syntax: `:::card` ... `:::`

3. **Component CSS**:
   - Each template styles components differently
   - Editorial: subtle borders, warm tones
   - Dark: high contrast, glows
   - Minimal: borders only, no shadows
   - Neon: gradients, glows, glass

4. **Test document**:
   - Create test-components.md with all components

## Success Criteria
- Cards render in all 4 templates with distinct styles
- Tags show color variants
- Metrics display large numbers
- Progress bars show percentage fill
