# Plan 02: Lucide Icons Integration

**Phase:** 2 - Visual System
**Goal:** Enable inline icons in markdown via :icon-name: syntax

## Steps

1. **Create src/lib/icons.ts**:
   - Fetch Lucide icons from CDN (unpkg or esm.sh)
   - Parse `:icon-name:` syntax in markdown
   - Replace with inline SVG
   - Support size modifiers: `:icon-name-lg:`, `:icon-name-sm:`

2. **Integrate in parser.ts**:
   - Post-process HTML to replace icon placeholders
   - Before template rendering

3. **Add icon styles to CSS**:
   - Inline alignment
   - Color inheritance
   - Size variants (sm: 16px, md: 24px, lg: 32px)

## Success Criteria
- `:home:` → inline SVG home icon
- `:arrow-right-lg:` → large arrow icon
- Icons inherit text color
- Works in all templates
