# Plan 01: Complete Template CSS System

**Phase:** 2 - Visual System
**Goal:** Build full CSS for all 4 templates with proper design systems

## Steps

1. **Create src/lib/palettes.ts** - Color palette definitions
   - 6 palettes per template
   - CSS custom properties generator

2. **Expand src/lib/template.ts**:
   - Editorial Warm: Playfair Display + Inter, earth tones
   - Dark Executive: Dark bg, amber/cyan accents
   - Clean Minimal: System fonts, pure Swiss design
   - Neon Gradient: Gradient text, purple/cyan theme

3. **Add typography scales**:
   - Editorial: Large serif headings, generous line-height
   - Dark: High contrast, bold weights
   - Minimal: Tight, precise
   - Neon: Gradient headings, glow effects

4. **Add component classes**:
   - `.card` - bordered/elevated containers
   - `.tag` - pill labels
   - `.metric` - big number displays
   - `.quote-block` - styled blockquotes
   - `.divider` - section separators

## Success Criteria
- All 4 templates render distinct visual styles
- Typography hierarchy is clear in each
- Colors apply consistently
